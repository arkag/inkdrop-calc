# Inkdrop Calculator

This simple plugin relies on MathJS to perform a basic evaluate() on anything between two carets.

## Usage

`^<math here>^`

Example:

| Markdown        | Output                 |
| --------------- | ---------------------- |
| `^1+1^`         | 2                      |
| `^sqrt(9)^`     | 3                      |
| `^hello world^` | Undefined symbol hello |
| `^sin(1)^`      | 0.8414709848078965     |
